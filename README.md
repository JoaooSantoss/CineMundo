# Cine-Mundo

Projeto Next.js (App Router) simples com CSS puro para replicar a tela de um cinema:
- Navbar
- Banner com bot√£o
- Lista de filmes

Como usar frontend :
1. cd frontend
2. Rode `npm install`
3. Rode `npm run dev`
4. Abra http://localhost:3000


como usar backend:
1. cd backend
2. instale os pacotes( 
    `npm install` --para a node-modules
    `npm install express`
    `npm install mysql2`
    `npm install mssql`
    `npm install body-parser`
    `npm install cors`
    `npm install dotenv`)
